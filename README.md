# Titanic Data Analysis 🚢

## 📌 Task 2 – Data Analysis Internship

This project analyzes the Titanic dataset to understand survival patterns.

## 🔍 Work Done

* Data cleaning (handled missing values)
* Survival analysis (gender, class, age)
* Data visualization using Seaborn & Matplotlib

## 📊 Key Insights

* Females had higher survival rate than males
* 1st class passengers had better survival chances
* Age influenced survival trends

## 🛠 Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Google Colab

## 📁 Dataset

Titanic Dataset from Kaggle
